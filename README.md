# Blackout-X-Perf
Module Magisk/KernelSU yang dirancang untuk mengoptimalkan performa perangkat Android secara otomatis. Modul ini mendeteksi saat sebuah game sedang berjalan dan mengaktifkan serangkaian *tweak* performa agresif. Saat game ditutup, semua pengaturan akan dikembalikan ke kondisi normal untuk menghemat baterai dan menjaga stabilitas sistem.
